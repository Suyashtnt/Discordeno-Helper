# Get ready

Discordeno Helper is very easy to get started with. There are 7 steps

1. Create github account
2. Go to the [Template Github](https://github.com/Suyashtnt/discordeno-helper-template)
3. Click "Use this template"
4. Fill out the prompts
5. Clone it using whatever way you like
6. Open it in your preferred IDE
7. Start coding!

For a more in-depth look on how to set it up, check out the [step-by-step](/step-by-step) guide
